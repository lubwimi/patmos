== Description ==

Easily remove or replace Tesseract footer credit. Just activate the plugin and the footer credits are removed. You can also enter text to add your own footer credits. This plugin provides a much easier method than having to modify PHP code which can break your site and has the downside of having to modify code each time your them is updated.

Go [here](https://upwerd.com/remove-tesseract-footer-credit) for step by step instructions with screenshots and video. You can also ask questions in the comment section or send me a message directly.

== Installation ==

1. From WordPress Admin, go to Plugins -> Add New
2. Search for plugin "Remove Tesseract Footer Credit"
3. Install and Activate
4. Go to Tools -> Remove Tesseract Footer Credit
5. Enter text you would like to change footer credits to or nothing to remove
7. Click Save
